Selamat Datang di BangronDB!

BangronDB adalah database NoSQL berbasis SQLite dengan API mirip MongoDB. Wizard ini akan membantu Anda mengkonfigurasi instalasi dalam beberapa langkah mudah.

Cepat & Ringan

Zero-config, berbasis SQLite

Enkripsi AES-256

Keamanan tingkat enterprise

API MongoDB-like

Sintaks familiar & mudah

Sebelum Memulai

Pastikan Anda memiliki akses tulis ke direktori instalasi dan PHP 8.0+ sudah terpasang dengan ekstensi PDO SQLite.
