Quick start guide

Mulai cepat dengan BangronDB

Gunakan pola eksplisit: buat database dan collection terlebih dahulu, lalu pilih resource yang sudah ada untuk query dan CRUD.

1. Install

composer require herdianrony/bangrondb

2. Buat resource

$client->createDB('app');
$client->createCollection('app', 'users');

3. Query dan CRUD

use BangronDB\Client;

$client = new Client(__DIR__ . '/data');
$client->createDB('app');
$client->createCollection('app', 'users');

$users = $client->selectCollection('app', 'users');

$id = $users->insert([
    'name' => 'John Doe',
    'email' => 'john@example.com',
    'role' => 'admin',
]);

$user = $users->findOne(['_id' => $id]);
$users->update(['_id' => $id], ['$set' => ['role' => 'editor']]);
$users->remove(['_id' => $id]);

Migration note

`selectDB()` dan `selectCollection()` sekarang non-lazy. Jika sebelumnya Anda mengandalkan pembuatan implicit, pindahkan ke `createDB()` dan `createCollection()`.
